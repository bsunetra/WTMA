Web Technology & Mobile Application

Subject Code: AI23431 | L-T-P-C: 1-0-4-3

Artificial Intelligence & Data Science
Artificial Intelligence & Machine Learning

Course Objectives

  • Provide foundational knowledge and practical skills in creating web pages using HTML and CSS.
  • Understand client-side scripting and DOM manipulation.
  • Implement server-side scripting techniques.
  • Understand Android application design concepts.
  • Gain basic Android application development skills.

Course Units

Unit I – Website Basics

Internet, HTTP, HTML5.2, CSS3.0 fundamentals.

Unit II – JavaScript

DOM, events, JSON, AJAX.

Unit III – Server Side Programming

Servlets, JSP, JDBC.

Unit IV – Mobile Application Development

Android fundamentals and components.

Unit V – ANDROID USER INTERFACE DESIGN

User Interface Screen elements,Designing User Interfaces with Layouts,Drawing and Working with Animation. Testing Android applications,Publishing Android application.

Course Outcomes

  • Design structured web pages.
  • Implement dynamic client-side logic.
  • Build simple server-side applications.
  • Create basic mobile apps.
  • Understand mobile programming concepts.
